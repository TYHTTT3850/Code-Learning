事件的类型

鼠标事件

定义：用户通过鼠标设备与页面进行交互时触发的事件，包括点击、移动、进入、离开、按下、释放等操作。

常见的事件：

  • click：点击(按下再释放)元素时触发。
  • dblclick：快速双击触发。
  • mousedown / mouseup：按下 / 释放鼠标按钮时触发。
  • mouseover / mouseout：鼠标指针进入 / 离开元素时触发。
  • mousemove：鼠标在元素上移动时持续触发。
  • contextmenu：右键点击(通常用于弹出菜单)。

键盘事件

定义：键盘事件用于监听用户对键盘的操作，主要包括按下、释放等动作，适用于表单输入、快捷键响应等场景。

常见的事件：

  • keydown：某个键被按下时触发(可能重复触发)。
  • keyup：键被释放时触发。
常见的属性：
  • e.key：返回被按下的字符(如 "Enter"、"a")
  • e.code：返回物理键位(如 "KeyA"、"ArrowUp")

焦点事件

定义：页面元素(通常是输入框、按钮等)获得或失去输入焦点时发生的事件。

常见的事件：

  • focus：元素获得焦点时触发。
  • blur：元素失去焦点时触发。
注意：这些事件不会冒泡，但有对应的冒泡版本 focusin 和 focusout。

文本框输入事件

定义：用户对输入框、文本框等可编辑区域输入内容时所触发的事件。

常见的事件：

  • input：用户输入内容时实时触发(比 change 更即时)。
  • change：当输入内容发生改变并失去焦点时触发。
  • compositionstart / compositionend：中文拼音输入过程中的事件。
