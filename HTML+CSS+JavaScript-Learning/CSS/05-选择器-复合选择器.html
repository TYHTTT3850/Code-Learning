复合选择器适用于网页有多个同名标签，每个标签都要差异化设置样式的情况

后代选择器

选择某元素的后代元素(所有后代，如儿子级别，孙子级别)

选择器写法：父选择器 子选择器{CSS属性}

示例：

这是span标签
这是div的儿子span

这是孙子span

子代选择器

只选择儿子这一级，不选择孙子及更后一级

写法：父选择器 > 子选择器{CSS 属性}

示例：

这是strong的儿子div
这是strong的孙子div

并集选择器

选中多组标签设置同一个样式

写法：选择器1 , 选择器2 , ... , 选择器N{CSS属性}

注意最有一个选择器后无逗号。

示例：

下划线，删除线，倾斜

交集选择器

选中同时满足多个条件的标签

写法：选择器1选择器2

选择器之间直接连写，无任何符号。
如果选择器之间有标签选择器，则标签选择器要在最前面

示例：

既是p标签，又是类box
