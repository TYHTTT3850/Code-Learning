背景属性

CSS常见的背景属性如下：

    背景色：background-color
    背景图：background-image
    背景图平铺方式：background-repeat
    背景图位置：background-position
    背景图缩放：background-size
    背景图固定：background-attachment
    背景复合属性：background

添加背景图

背景图默认是平铺(复制)的效果，当设置的属性的的大小大于图片尺寸时，会复制图片以铺满所设置的大小

设置背景图的平铺方式

设置了不要复制图片，因此图片没有铺满整个矩形。

主要的平铺方式如下：

    repeat：默认，复制以铺满整个矩形
    no-repeat：不复制
    repeat-x：水平方向平铺
    repeat-y：垂直方向平铺

设置背景图位置

位置的属性可以填入关键字或具体的坐标：
图片的位置是相对于矩形左上角的位置，即左上角为原点(0,0)。往右或往下对应的坐标变大
关键字：

    left：左侧
    right：右侧
    center：居中
    bottom：底部
    top：顶部

坐标：
水平坐标(数字+px) 垂直坐标(数字+px)
数字正负都可以。
水平方向上：正数往右移动，负数往左移动
垂直方向上：正数往下移动，负数往上移动

背景图缩放

背景图缩放的常用设置方式：

    关键字
      cover：等比例缩放背景图片以完全覆盖背景区，可能导致部分图片看不见
      contain：等比例缩放背景图片使图片完全包含在背景区中，可能导致部分背景区留白
    百分比：根据背景区尺寸计算图片大小

背景图固定

背景图固定后，图片不会随着网页的滚动而滚动
