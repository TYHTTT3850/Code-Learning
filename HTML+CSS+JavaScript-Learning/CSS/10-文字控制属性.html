文字控制属性

文字控制属性主要有以下几种：

属性 描述
font-size 字体大小
font-weight 字体粗细
font-style 字体倾斜
line-height 行高
font-family 字体族
font 字体复合属性
text-indent 文本缩进
text-align 对齐方式
text-decoration 修饰线
color 颜色

字体大小

属性名：font-size
属性值：文字尺寸，PC 端网页最常用的单位是px，不写单位时，属性不生效。

字体粗细

属性名：font-weight
属性值：数字(正常400，加粗700)，关键字(正常normal，加粗bold)。实际开发中，数字更实用。

字体是否倾斜

属性名：font-style
属性值：normal(不倾斜)，italic(倾斜)。倾斜在实际中不怎么用。

字体族

属性名：font-family
属性值：字体名

示例：

更改字体
font-family可以填入多个属性值，属性值之间用 , 隔开。使用的优先顺序是从左向右递减。
font-family 属性最后可以设置一个字体族名以防止前面的字体在电脑上都没有，网页开发建议使用无衬线字体(sans-serif)

font符合属性

一般用在网页文字的公共样式

一个属性对应多个值，每个值之间用空格隔开

属性从左往右分别是：是否倾斜 是否加粗 字号/行高 字体(必须按顺序书写)。
同样，若行高仅填入数字，则行高为当前标签的字体大小的倍数
字号和字体值必须书写，否则 font 属性不生效。

文本缩进

属性名：text-indent
属性值：数字+px，数字+em(常用)

文本对齐

属性名：text-align
属性值：left(左对齐，默认)、center(居中)、right(右对齐)

示例：

居中对齐

text-align本质控制的是内容的对齐方式，要在父级标签设置

要在 img 标签的上一级 div 标签中设置

修饰线

属性名：text-decoration
属性值：none、underline(下划线)、line-through(删除线)、overline(上划线)

文字颜色

属性名：color
属性值：颜色
颜色表示方法：颜色的英文单词(学习用)、RGB表示法(了解)，RGBA表示法(常用，实现透明色)，十六进制表示法(常用，从设计稿复制)
