基本使用

table 嵌套 tr，tr 嵌套 td/th。

tr 表示行，有几行写几个。th 表示表头(字段名)，一般在表格的第一行使用，会有加粗和居中效果。td 表示表格的具体内容

表格默认没有边框线，使用 border 属性添加。

表格结构标签

单元格合并

有些单元格的信息可能相同，因此可以合并

单元格的合并有跨列合并和跨行合并两种

合并单元格在 td 标签中添加属性，rowspan 表示跨行，colspan 表示跨列，要合并几个就填几个。
