1.什么是 DOM？

DOM 是浏览器将 HTML 转换为 JavaScript 可操作对象的模型。你可以像操作树形结构一样读取、修改、删除页面内容。

2.获取元素的方式

  • document.getElementById(id)：通过 ID 获取单个元素
  • document.querySelector(selector)：通过 CSS 选择器获取第一个匹配的元素
  • document.querySelectorAll(selector)：获取所有匹配元素，返回 NodeList
  • document.getElementsByClassName(name)：获取指定类名的元素集合
  • document.getElementsByTagName(name)：获取指定标签名的元素集合

2.修改文本内容

  • element.textContent：设置或获取元素的文本内容。例如：element.textContent = "新文本";
  • element.innerHTML：设置或获取元素内部的 HTML 内容(不包括当前标签本身)。例如：element.innerHTML = "新HTML";
  • element.outerHTML：设置或获取元素的 HTML 内容(包括当前标签本身)。例如：element.outerHTML = "

3.修改样式

  • element.style.css属性：设置单个 CSS 属性。例如：element.style.color = "blue";
  • element.style.cssText：设置多个 CSS 属性。例如element.style.cssText = "color: red;";

4.操作类名

  • element.classList.add(className)：添加一个或多个类名。例如：element.classList.add("active");
  • element.classList.remove(className)：移除指定的类名。例如：element.classList.remove("active");
  • element.classList.toggle(className)：切换类名，若存在则移除，否则添加。例如：element.classList.toggle ("highlight");
  • element.classList.contains(className)：检查是否包含指定类 element.classList.contains("active");

5.操作结构

  • document.createElement(tagName)：创建一个新的 HTML 元素。例如：const newDiv = document.createElement("div");
  • element.appendChild(child)：将子元素添加到指定元素的末尾。例如：element.appendChild(newDiv);
  • element.insertBefore(newNode, referenceNode)：在指定元素前插入新元素。例如：element.insertBefore(newDiv, referenceElement);
  • element.removeChild(child)：删除指定的子元素。例如：element.removeChild(newDiv);
  • element.replaceChild(newNode, oldNode)：替换子元素。例如：element.replaceChild(newDiv, oldDiv);

6.操作属性

  • element.setAttribute(name, value)：设置指定元素的属性。例如：element.setAttribute("id", "newId");
  • element.getAttribute(name)：获取指定元素的属性值。例如：element.getAttribute("id");
  • element.removeAttribute(name)：移除指定元素的属性。例如：element.removeAttribute("id");
