# `which` 命令

通过 `which` 命令，查看所使用的一系列命令的程序文件存放在哪。语法：

```bash
which {要查找的命令}
```

# `find` 命令

通过 `find` 命令查找指定的文件。语法：

```bash
find {起始路径} -name "被查找的文件名"

find {起始路径} -size {+|-}n[k | M | G | T]
```

`-name` ：表明以文件名模式查找。

`-size` ：表明以大小模式查找。

`+|-` ：`+` 表大于，`-` 表小于。

`n` ：大小(数字)。

`k | M |G` ：表示大小单位，k(小写字母)表示KB，M表示MB，G表示GB，T表示TB。

若查找时间过长，按 ctrl+c 强制终止。

`find` 文件名查找模式同样支持通配符 `*` 查找，与删除命令 `-rm` 一样。

**回顾**：

`*` 表示匹配任意内容(包含空)，例如：

test*表示任何以test开头的内容。

*test表示任何以test结尾的内容。

\*test\*表示任何包含test的内容。
