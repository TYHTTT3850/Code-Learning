# 进程

程序在运行时，会被操作系统注册为系统中的一个进程，并未其分配一个独有的进程ID(进程号)

# `ps` 命令查看进程

语法：

```bash
ps [-e] [-f]
```

`[-e]` ：显示出全部的进程。

`[-f]` ：以完全格式化的形式展示信息(展示全部信息)

一般情况下，固定使用：

```bash
ps -e -f
```

从左到右分别是：

- UID ：进程所属的用户ID。

- PID ：进程的进程号ID。

- PPID ：进程的父ID(启动此进程的其他进程)。

- C ：此进程的CPU占用率。

- STIME ：进程的启动时间。

- TTY ：启动此进程的终端序号，若显示为"?"，则表示非终端启动。

- TIME ：进程累计占用CPU的时间。

- CMD ：进程对应的名称或启动命令或启动路径。

`ps` 命令配合管道符 `|` 和 `grep` 过滤命令，可以准确的查找出指定的进程的信息。
