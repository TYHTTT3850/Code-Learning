**内存：** 软件在运行时，临时存储数据。

**内存地址：** 操作系统为了更快地查询内存中的数据，会把内存以字节为单元划分为很多个小区域，一个区域就是一个字节，这些小区域都有自己独立的编号，编号就是内存地址。

**内存地址编号的规则：**

- 32位操作系统中，内存地址以32位的二进制表示，地址范围从 `0000 0000 0000 0000 0000 0000 0000 0000` —— `1111 1111 1111 1111 1111 1111 1111 1111`(2的32次)。也就是最大支持 4GB 的内存。
- 64位操作系统中，内存地址以64位的二进制表示，最大内存地址为 2^64，最大支持 17179TB 的内存。64 位系统如果直接以二进制显示内存地址，会显得十分冗长，所以会以每四位一组，转换为 16 进制，方便阅读。
