## 子类能继承什么

**构造方法**：不能被子类继承，但可以用 `super` 关键字调用。

**成员变量**：可以被子类继承，`private` 修饰的也可以，但是无法直接调用。

**成员方法**：虚方法可以继承。`final` 和 `static` 修饰的不能被继承，但可以调用。`private` 修饰的不能继承也不能调用。

## 虚方法是什么

就是普通的成员方法，非 `final` 、`static` 、`private` 修饰的方法。

**虚方法的继承规则**：父类把所有的虚方法的内存地址汇总起来，形成成虚方法表。子类继承时会拿到父类的虚方法表，并在此基础上添加自己的虚方法。

**方法重写的本质**：替换被重写的虚方法的内存地址。

## `final` 修饰的方法

**编译期**：一级一级往上找，遍历整个继承链，确定调用的方法在A类中，记录该方法的地址。

**运行期**：直接运行编译时确定的方法，虽然降低了编译的速度，但却保证了运行的速度。

## `static` 修饰的方法

**编译期**：一级一级往上找，遍历整个继承链，确定调用的方法在A类中，将对象调用方法直接修改为类名调用方法。

**运行期**：直接运行。
