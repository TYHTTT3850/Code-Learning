静态方法只能访问静态变量和其他的静态方法。

非静态方法可以访问静态变量或静态方法，也可以访问非静态的成员变量和非静态的成员方法。

静态方法中是不能出现 this 关键字的。

原因：静态是随着类的加载而加载的，类加载好了，对象不一定创建好了，所以静态方法只能访问静态变量和其他的静态方法。

## 重新认识 main 方法

`public static void main(String[] args){}`

- `public` ：被 JVM 调用，访问权限最大。
- `static`：被 JVM 调用，类名访问。
- `void`：不需要给 JVM 返回值。
- `main` ：固定的名字，被 JVM 识别。
- `String[] args` ：用于接收运行的参数。
