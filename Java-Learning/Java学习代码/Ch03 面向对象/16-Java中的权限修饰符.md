## Java 权限修饰符访问范围

| 修饰符                | 同类     | 同包       | 不同包子类 | 不同包非子类 |
| --------------------- | -------- | ---------- | ---------- | ------------ |
| public                | 可以访问 | 可以访问   | 可以访问   | 可以访问     |
| protected             | 可以访问 | 可以访问   | 可以访问   | 不可以访问   |
| 默认(package-private) | 可以访问 | 可以访问   | 不可以访问 | 不可以访问   |
| private               | 可以访问 | 不可以访问 | 不可以访问 | 不可以访问   |

## 修饰符可作用对象

| 修饰符 | 顶级类 | 成员变量 | 方法 | 构造方法 | 内部类 |
|---|---|---|---|---|---|
| public | 可以使用 | 可以使用 | 可以使用 | 可以使用 | 可以使用 |
| protected | 不可以使用 | 可以使用 | 可以使用 | 可以使用 | 可以使用 |
| 默认 | 可以使用 | 可以使用 | 可以使用 | 可以使用 | 可以使用 |
| private | 不可以使用 | 可以使用 | 可以使用 | 可以使用 | 可以使用 |
