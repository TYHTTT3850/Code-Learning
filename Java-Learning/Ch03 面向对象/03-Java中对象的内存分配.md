## 对象的创建

Java 中创建对象的七步，以 `Student stu = new Student();` 为例：

- 1、加载类的class字节码文件。

- 2、声明等号左边的局部变量。

- 3、在堆里面开辟一个空间(对象)。

- 4、给对象里的属性进行默认初始化。

- 5、给对象中的属性进行显式初始化。

- 6、给对象中的属性利用构造方法初始化。

- 7、把对象的内存地址赋值给等号左边的变量。

方法出栈以后，方法里面的变量全部消失。

如果没有任何地方使用堆里面的对象，那么对象也会从堆里面消失。

方法区的字节码信息一般不会消失，除非关闭虚拟机。

## 对象在方法中的传递

把一个对象传递给方法，实际上传递的是对象的内存地址。

所以当多个变量指向一个对象的时候：只要有一个变量修改了对象中的属性，其他变量访问时就是修改之后的结果了。

## this 关键字的本质

this 代表的是所在方法调用者的地址值
